Frequently Asked Questions

How do I add domains to the whitelist?

Enter one domain per line in the whitelist textarea. You can use wildcards (*) to match multiple subdomains.

What are status indicators?

Status indicators show the current state of the extension for each website in your browser toolbar.

How do I report a bug?

You can report issues through our GitHub repository or contact support directly.
